# UI revision 4 — task pack

Блок `25.1–25.7` выполняется поверх закрытых блоков `21–24` и не меняет онтологию проекта.

Опорный файл требований: `codex/reference/UI_REVISION_4_CHECKED_REQUIREMENTS.md`.

## Состав блока

| № | Тема |
|---|---|
| 25.1 | Консольные команды БД через command registry |
| 25.2 | DXF dialog: нижняя toolbar-панель, режимы окраски и повороты preview |
| 25.3 | DXF model tab: layout 5:4, удаление служебной подсказки, исправление цветов preview |
| 25.4 | DXF profiles tab: табличная разметка, палитра 4×4, стандарты/типы/профили/материалы |
| 25.5 | DXF logs tab: компактные логи и таксономия `error/warning/info` |
| 25.6 | Диалог `Ветер`: локализация, поля 40 px, DTO режимов `simple/sp20/pue` |
| 25.7 | ProjectTree/PropertiesPanel: строки 40 px, AutoCAD-like properties и multiselect через Shift |

## Общие проверки

После каждой frontend-задачи:

```bash
cd frontend && npm run build
cd frontend && npm run lint
```

## Инварианты

- Команды меню, hotkeys и консоль используют единый `command registry`.
- Внутренние enum/API-ключи остаются латиницей; русские тексты идут через локализацию/display-name map.
- DXF preview не изменяет основную модель до подтверждения импорта.
- Ошибки импорта блокируют импорт; предупреждения импорт не блокируют; информационные сообщения не окрашивают модель в желтый.
- Выбор профиля предшествует выбору материала; материалы фильтруются по толщине профиля.
