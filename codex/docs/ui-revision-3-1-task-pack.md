# UI revision 3.1 task pack

## Назначение

Пакет добавляет блок задач `23.1–23.5` поверх выполненной ревизии 3 (`22.1–22.6`) без изменения онтологии проекта.

## Состав

1. `23.1` — перестроить DXF import dialog в структуру: шапка → tabs `Модель | Профили | Логи` → нижний блок кнопок.
2. `23.2` — добавить preview-нормализацию DXF: `Z >= 0`, ось `Z` по центру модели, повороты вокруг осей и reset.
3. `23.3` — оформить вкладку `Профили` как таблицу `Цвет | Тип профиля | Стандарт | Профиль`; цвет — swatch с палитрой.
4. `23.4` — сделать вкладку `Логи` компактной, агрегированной и без raw JSON.
5. `23.5` — очистить диалог `Ветер` и добавить документацию/checklist ревизии 3.1.

## Главные ограничения

- Backend, расчетное ядро и БД не трогать.
- Служебную информацию не возвращать в основной UI.
- Preview не должен менять основную модель до нажатия `Импорт`.
- Не менять модель ветра: `direction` остается 3D-вектором, UI сохраняет `z = 0`.
- Тип/стандарт профиля являются фильтрами выбора существующего профиля, а не новой онтологией модели.

## Проверка после каждой задачи

```bash
cd frontend && npm run build
cd frontend && npm run lint
```
