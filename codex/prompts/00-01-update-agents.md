# Task 0.1 — Update AGENTS.md

## Контекст
Проект GridEng. Текущий фокус — frontend приложения для расчета стержневых металлоконструкций с прицелом на опоры ЛЭП свыше 35 кВ.

## Задача
Обнови `AGENTS.md` для проекта GridEng.

Добавь правила:
1. Основной текущий фокус — frontend приложения для расчета стержневых металлоконструкций.
2. `backend/app/lib` с расчетным ядром не менять без отдельного указания.
3. БД и постоянные backend-контракты пока не реализовывать, кроме явно указанного контракта API характеристик сечений.
4. Frontend архитектура: `app/layout`, `entities/model`, `entities/section`, `features`, `widgets`, `shared`.
5. DXF импорт v0.1 на первом этапе поддерживает только `LINE`, coordinates, `color` / `colorIndex` / `trueColor`, `layer`, `handle`. Остальные primitives игнорируются.
6. Внутренняя модель должна импортироваться/экспортироваться в JSON.
7. После каждой frontend-задачи, которая меняет frontend-код, запускать `cd frontend && npm run build` и `cd frontend && npm run lint`.

## Ограничения
Не меняй код приложения. Только `AGENTS.md`.

## Отчет
В конце покажи summary, измененные файлы, команды, результат проверок, риски.
