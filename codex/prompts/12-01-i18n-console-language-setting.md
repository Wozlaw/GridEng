# Task 12.1 — Ввести i18n-слой ru/en и UI-настройку языка

## Цель

Централизовать пользовательские строки и подготовить переключение языка через консоль без добавления переключателя в меню.

## Сделать

1. Создать `frontend/src/shared/i18n/`:
   - `types.ts`;
   - `ru.ts`;
   - `en.ts`;
   - `index.ts`;
   - простой `t(key, params?)` или hook `useI18n()`.
2. Добавить UI-настройки приложения отдельно от расчетной модели:
   - `language: 'ru' | 'en'`;
   - хранение в `localStorage`.
3. Перевести основные видимые строки:
   - режимы просмотра;
   - top menu/ribbon labels/tooltips;
   - project tree headings;
   - properties headings;
   - status bar;
   - alerts;
   - console help labels.
4. Пока не обязательно перевести абсолютно все технические строки, но новые UI-компоненты должны использовать i18n сразу.
5. Добавить будущую консольную команду в command registry/заглушку, если registry еще не готов:
   - `lang ru`;
   - `lang en`;
   - `settings.set language ru|en`.

## Ограничения

- Не добавлять переключатель языка в меню.
- Не тащить тяжелую i18n-библиотеку, если простой словарь достаточен.

## Проверка

- По умолчанию язык русский.
- После смены языка значение сохраняется в localStorage.
- `npm run build` и `npm run lint` проходят.
