# Task 8.2 — Local profile catalog

## Задача
Реализуй временный локальный каталог профилей в `frontend/src/entities/section/profileCatalog.ts`.

## Требования
- Несколько уголков `L`.
- Несколько труб `pipe`.
- Несколько прямоугольных профилей `rect`.
- Значения допускается указать приблизительно или оставить undefined, если нет точных данных.
- API: `getProfileCatalog()`, `findProfileById(id)`, `createCustomProfile(input)`.
- Не подключать БД/backend.

## Проверки
- `cd frontend && npm run build`
- `cd frontend && npm run lint`
