# Task 25.4 — DXF profiles tab: table, palette, profile search and materials

## Цель

Доработать вкладку `Профили` DXF dialog до табличного назначения `цвет → тип/стандарт/профиль/материал` с поиском профиля, выбором материала и контрастной палитрой.

## Изменения

1. Сделать строки вкладки `Профили` визуально табличными: разделители строк, единая сетка, высота строк/controls 40 px.
2. Рекомендуемая структура колонок:
   - `Цвет` — фиксированная ширина 56–72 px;
   - `Тип профиля`;
   - `Стандарт`;
   - `Профиль`;
   - `Материал`.
3. Все поля, кроме `Цвет`, выровнять по ширине и высоте.
4. `Цвет` показывать только как цветовой индикатор/swatch.
5. По клику на swatch открывать палитру 4×4. Цвета брать из MUI palette/color set по аналогии с документацией MUI.
6. Палитра должна предлагать контрастные цвета и не показывать цвета, уже назначенные другим профилям/группам.
7. Изменение цвета допускается как изменение текущего ключа/цвета группы назначения; связь с исходной DXF-группой сохранять не обязательно.
8. Типы профилей отображать на русском языке через display-name/localization map. Внутренние enum/API-ключи не менять. Учесть переключение языков.
9. При изменении стандарта:
   - пересчитать доступные типы профиля;
   - если текущий тип стал невалидным, выбрать первый доступный тип;
   - пересчитать доступные профили;
   - если текущий профиль стал невалидным, выбрать первый профиль из детерминированно отсортированного списка.
10. Сортировка профилей обязательна по размеру/обозначению так, чтобы `первый из списка` был стабильным.
11. В поле `Профиль` добавить поиск/autocomplete по `designation`, `display_name`, `standard`, `profile_type` в рамках выбранного стандарта и типа.
12. Добавить выбор материала. Материал хранить как `material_id`, отображать русским названием/маркой.
13. При смене профиля фильтровать материалы по толщине/типу проката. Если текущий материал подходит — сохранять выбор. Если не подходит — выбрать первый доступный материал.
14. Если для профиля/толщины нет подходящих материалов, строка становится ошибочной, а импорт блокируется до назначения материала.
15. Убрать надписи `Назначьте профиль для этой группы.` и прочие пояснения из вкладки; статус строки показывать компактно, подробности — в логах.

## Acceptance criteria

1. Вкладка визуально похожа на таблицу.
2. Палитра 4×4 открывается по клику на swatch.
3. Цвета в палитре не дублируют уже назначенные цвета и остаются контрастными.
4. Типы профилей локализованы без изменения API-контракта.
5. Смена стандарта/типа/профиля детерминированно обновляет зависимые поля.
6. Профиль можно искать.
7. Материал выбирается и фильтруется по толщине; импорт блокируется без валидного материала.
8. Пройдены build/lint.
