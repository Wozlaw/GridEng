# Общие правила выполнения

- Работай маленьким diff. Не пытайся выполнить соседние задачи из следующих файлов.
- Перед правками дай краткий план и перечисли файлы, которые планируешь менять.
- Проект уже выполнен до задачи 9.2 включительно. Не откатывай реализованные `loads/moments rendering` и `restraints rendering`, а адаптируй их к новой модели.
- Frontend: `frontend/`, React + TypeScript + Vite + MUI + Three/R3F + Zustand.
- Backend и БД не трогать. Нормативный расчет ветра не реализовывать. API пересчета ЛСК/ГСК не реализовывать в этой ветке.
- После каждой задачи запусти: `cd frontend && npm run build` и `cd frontend && npm run lint`.
- В конце ответа покажи: summary, измененные файлы, команды, результат build/lint, риски/техдолг.


# Task 13.2 — Панели, Status bar, visibility toggles и Fit

## Цель

Доработать CAD layout: дерево проекта и Properties должны менять ширину/сворачиваться, а служебные счетчики должны уйти в status bar.

## Сделать

1. ProjectTree panel:
   - resizable width `240–420 px`;
   - collapsible to hidden state;
   - кнопка сворачивания/раскрытия по центру границы со сценой;
   - состояние в localStorage.
2. Properties panel:
   - resizable width `280–520 px`;
   - collapsible;
   - кнопка по центру границы со сценой;
   - состояние в localStorage.
3. Status bar:
   - заменить чипы строкой:
     `Проект: {name} / Ноды: {n} / Элементы: {m} / Нагрузки: {l} / Закрепления: {r} / Профили: {p} / Материалы: {mat} / Статус: {valid|ошибки|предупреждения}`;
   - справа можно показывать `Выбрано: ...`;
   - убрать `DXF tolerance` из status bar/меню, перенести в настройки импорта/проекта.
4. Visibility state:
   - добавить toggles для осей, сетки, нагрузок, закреплений;
   - команды должны быть доступны из ribbon и command registry.
5. Fit:
   - убрать кнопку Fit со сцены;
   - разместить команду в `Режим просмотра`;
   - hotkey не должен конфликтовать с браузерным поиском; предпочтительно `F`, не `Ctrl+F`;
   - selection не вызывает Fit.
6. Служебную debug-информацию убрать из основного UI.

## Проверка

- Ширина панелей сохраняется после reload.
- Панели можно скрыть и вернуть.
- Оси/сетка скрываются и показываются.
- Fit работает только по команде пользователя.
- `npm run build` и `npm run lint` проходят.
