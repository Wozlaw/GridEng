# Task 25.3 — DXF model tab layout 5:4 and preview colors

## Цель

Очистить вкладку `Модель` DXF dialog от служебных подсказок, настроить viewport 5:4 и исправить отображение цветов DXF-групп на 3D preview.

## Изменения

1. Убрать из основного UI служебное поле/подсказку: `Выберите .dxf файл, чтобы построить предпросмотр импорта.`
2. Если файл не выбран, допускается минимальное empty-state состояние внутри вкладки `Модель`, но без отдельного служебного блока и без лишней технической информации.
3. Сцена 3D preview занимает все доступное пространство вкладки `Модель` между tabs/header и нижней toolbar-панелью.
4. Соотношение canvas/viewport — `5:4`.
5. Высота viewport берется из доступной высоты после удаления лишних элементов; ширина вычисляется как `height * 5 / 4`.
6. Высота модальной формы остается прежней; ширина модальной формы определяется шириной 3D-сцены плюс необходимый dialog chrome.
7. Для малых экранов добавить responsive fallback: не выходить за viewport, не ломать toolbar и не создавать горизонтальный скролл.
8. Сцену отдельно в рамку не помещать. Для контейнера 3D-сцены установить `margin: 0`, `padding: 0`. Не распространять это правило на всю модальную форму.
9. Исправить проблему: при импорте цвета не отображаются на модели 3D-сцены. В режиме `Окраска: профили` элементы должны использовать цвет DXF-группы/назначенный display color.
10. В режиме `Окраска: диагностика` цвета должны соответствовать таксономии статусов из задачи 25.5.

## Acceptance criteria

1. Служебная подсказка удалена.
2. Viewport имеет соотношение 5:4 и не имеет отдельной рамки.
3. Dialog не выходит за экран на малых разрешениях.
4. Цвета DXF-групп видны на preview в режиме `Окраска: профили`.
5. Пройдены build/lint.
