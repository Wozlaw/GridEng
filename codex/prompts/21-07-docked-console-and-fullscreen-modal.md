# Task 21.7 — Консоль: нижняя docked-панель и fullscreen modal

## Цель

Перевести консоль из чисто модального интерфейса в нижнюю resizable-панель между боковыми панелями, сохранив текущий механизм команд и истории.

## Архитектурное требование

Не переписывать консоль с нуля. Вынести общий компонент, например:

- `ConsoleView`;
- `ConsoleContent`;
- или аналогичное имя по текущей структуре.

Этот общий компонент должен использоваться:

1. в docked-панели в нижней части центральной области;
2. в fullscreen modal.

## Docked-консоль

1. Расположение: нижняя часть экрана между `ProjectTreePanel` и `PropertiesPanel`.
2. Не перекрывает боковые панели.
3. Верхний край resizable.
4. Интерфейс минимальный:
   - окно вывода консоли;
   - ниже строка ввода команды;
   - в начале строки иконка консоли;
   - в конце кнопка ввода команды;
   - рядом кнопка развернуть.
5. Лишние кнопки и поля убрать из основного UI.
6. Очистку логов и прочие сервисные действия перенести в служебные команды консоли.

## Fullscreen modal

1. Открывается кнопкой развернуть из docked-консоли или командой `Консоль` из ribbon.
2. Высота — почти весь экран/на всю доступную высоту.
3. Интерфейс тот же минимальный `ConsoleView`.
4. Допускается только кнопка закрытия диалога (`X`) сверх минимального набора.

## Ограничения

- Не ломать существующие команды `help`, debug/settings/source info/service commands.
- Не дублировать логику выполнения команд между modal и docked.
- Не использовать `window.alert`.

## Ожидаемые места изменений

- `frontend/src/widgets/command-console/*`
- `frontend/src/app/layout/*` или `CadShell`
- `frontend/src/app/commands/*` при необходимости

## Критерии готовности

- Консоль доступна в нижней центральной области.
- Верхний край панели изменяет высоту консоли.
- Fullscreen modal использует тот же компонент отображения/ввода.
- История и выполнение команд работают одинаково в обоих режимах.

## Проверка

```bash
cd frontend
npm run build
npm run lint
```

Ручная проверка: выполнить `help`, сервисную команду очистки логов, открыть/закрыть fullscreen console.
