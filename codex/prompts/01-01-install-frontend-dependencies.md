# Task 1.1 — Install frontend dependencies

## Задача
Добавь при необходимости frontend-зависимости:

Runtime:
- `zod`
- `zustand`
- `@mui/material`
- `@emotion/react`
- `@emotion/styled`
- `@mui/icons-material`
- `three`
- `@react-three/fiber`
- `@react-three/drei`
- `dxf-parser`

Dev:
- `@types/three`

Команды:
```bash
cd frontend
npm install zod zustand @mui/material @emotion/react @emotion/styled @mui/icons-material three @react-three/fiber @react-three/drei dxf-parser
npm install -D @types/three
npm run build
npm run lint
```

## Ограничения
Не менять бизнес-логику. Не добавлять UI. Не менять backend.
