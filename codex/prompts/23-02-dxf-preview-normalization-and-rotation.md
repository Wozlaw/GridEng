# Task 23.2 — DXF preview normalization and rotation controls

## Цель

Добавить нормализацию координат preview-модели DXF и управляемый поворот вокруг осей без изменения онтологии основной модели.

## Требования

1. При загрузке DXF preview выполнить нормализацию координат для отображения:
   - минимальная `Z` координата preview должна стать `0`;
   - ось `Z` должна проходить через центр модели в плане `XY`;
   - модель должна быть визуально центрирована в preview-сцене.
2. Нормализация должна применяться к preview transform/derived geometry, а не разрушать исходные DXF данные, необходимые для логов и диагностики.
3. Добавить возможность поворота preview-модели вокруг осей с последующей нормализацией координат.
4. Поворот должен быть пользовательским действием в DXF import dialog, но не должен применяться к основной модели до нажатия `Импорт`.
5. Минимальный UI:
   - компактные icon buttons или numeric controls для поворота вокруг `X`, `Y`, `Z`;
   - reset rotation;
   - tooltip/aria-label для каждого действия.
6. После изменения поворота автоматически пересчитать нормализованный bbox/preview transform.
7. В логах фиксировать примененные preview rotation/normalization параметры как summary, без JSON.
8. Если уже существует логика `force2DToXY`/`centerOnXY`, не ломать ее; новая нормализация должна быть совместима с текущими `DxfImportSettings`.

## Важное архитектурное правило

Разделить:

- `source/import geometry` — исходные координаты/цвета/handle из DXF;
- `preview display transform` — нормализация и поворот для просмотра;
- `final import transform` — то, что реально применяется при импорте в модель.

Если текущий код не разделяет эти понятия, сделать минимальный безопасный слой-адаптер для preview, не переписывая весь импорт.

## Ограничения

- Backend и расчетное ядро не трогать.
- Не менять формат сохраненной модели без отдельной миграционной задачи.
- Не смешивать нормализацию просмотра и структурную конвертацию DXF в GridEng model.

## Критерии готовности

- Preview после загрузки стоит на `Z >= 0`.
- Ось `Z` проходит через центр модели в плане.
- Повороты вокруг осей работают и не ломают preview diagnostics/profile coloring.
- Reset возвращает исходное положение preview.
- Build/lint проходят.
