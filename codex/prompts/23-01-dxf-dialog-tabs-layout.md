# Task 23.1 — DXF import dialog tabs layout

## Цель

Перестроить окно импорта DXF в компактный рабочий диалог с тремя вкладками: `Модель`, `Профили`, `Логи`.

## Требования

1. Сохранить шапку окна с наименованием операции/файла.
2. Сохранить нижний блок кнопок из ревизии 3:
   - выбрать/заменить файл;
   - открыть/перейти к логам, если кнопка еще нужна вне вкладок;
   - отмена;
   - импорт.
3. Все кнопки оставить icon button с `Tooltip` и `aria-label`.
4. Убрать из основной области все пояснения и всю лишнюю служебную информацию.
5. Всю оставшуюся область занять MUI `Tabs` / `TabPanel` с вкладками:
   - `Модель`;
   - `Профили`;
   - `Логи`.
6. Вкладка `Модель`:
   - 3D-сцена занимает все доступное пространство вкладки;
   - без текстовых пояснений вокруг сцены;
   - режимы отображения из ревизии 3 (`Диагностика` / `Профили`) должны остаться доступными, но компактно, например через icon toggle / segmented control поверх сцены или в узкой toolbar-строке.
7. Подобрать размер диалога так, чтобы 3D-сцена имела нормальное соотношение сторон:
   - не делать узкий вертикальный modal;
   - ориентир: `maxWidth="lg"`/`xl` или проектный fullscreen-ish dialog;
   - сохранить адаптивность для малых экранов.
8. Служебные summary/счетчики (`linesCount`, `nodesCount`, `membersCount`, `ignoredEntitiesCount`, `zRange`, `mergedNodesCount`, `danglingMembersCount`) не показывать на вкладках `Модель` и `Профили`; переносить во вкладку `Логи`.
9. Не менять момент применения импорта: preview-модель не должна попадать в основной `modelStore` до нажатия `Импорт`.

## Ограничения

- Не менять DXF parser и модель данных в этой задаче, кроме минимального props/state, необходимого для tabs layout.
- Не добавлять новые диагностики: они обрабатываются в соседних задачах.
- Не выводить raw JSON.

## Критерии готовности

- Окно DXF имеет структуру: шапка → tabs area → нижний button block.
- Основная область не содержит лишних пояснений и служебных метрик.
- 3D preview на вкладке `Модель` занимает все доступное пространство и визуально читается.
- Build/lint проходят.
