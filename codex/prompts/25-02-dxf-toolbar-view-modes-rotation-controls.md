# Task 25.2 — DXF dialog toolbar: view modes and rotation controls

## Цель

Перенести кнопки режимов окраски и поворота DXF preview в нижнюю toolbar-панель окна импорта, показывая их только на вкладке `Модель`.

## Контекст

Работать поверх задач `23.1–23.5` и `24.1–24.5`. DXF import dialog уже имеет вкладки `Модель | Профили | Логи`.

## Изменения

1. В нижней панели DXF dialog сформировать группы кнопок:
   - основные действия: выбор/замена DXF-файла, отмена, импорт;
   - режим окраски preview: `Окраска: диагностика`, `Окраска: профили`;
   - transform preview: поворот вокруг `X`, `Y`, `Z`, сброс.
2. Группы `view mode` и `rotation/reset` отделить от основных действий вертикальным разделителем.
3. Кнопки `Окраска: диагностика` / `Окраска: профили` показывать только на вкладке `Модель`; на вкладках `Профили` и `Логи` скрывать.
4. Кнопки поворота `X/Y/Z` и сброса показывать только на вкладке `Модель`; на вкладках `Профили` и `Логи` скрывать.
5. Все кнопки выполнить как icon buttons с `Tooltip` и `aria-label`.
6. Терминологически не смешивать вкладку `Профили` и режим окраски `Профили`; в tooltip использовать формулировку `Окраска: профили`.
7. Поворот по кнопке выполнять на `+90°` вокруг глобальной оси `X/Y/Z` по правилу правой руки, если текущая реализация не фиксирует другой знак.
8. Поворот применяется только к preview/import transform и не меняет основную модель до подтверждения импорта.
9. После каждого поворота выполнять нормализацию preview-модели согласно текущей логике: `Z >= 0`, модель центрирована по принятому правилу.
10. Кнопка сброса сбрасывает только preview transform/повороты/нормализацию, но не сбрасывает выбранные профили, материалы и настройки импорта.

## Acceptance criteria

1. На вкладке `Модель` доступны кнопки режимов окраски и поворота.
2. На вкладках `Профили` и `Логи` эти кнопки скрыты.
3. Вертикальные разделители визуально отделяют группы команд.
4. Повороты не изменяют основную модель до нажатия `Импорт`.
5. Сброс не удаляет назначения профилей/материалов.
6. Пройдены build/lint.
