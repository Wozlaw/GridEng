# Task 1.2 — Create frontend folder structure

## Задача
Создай структуру каталогов:

```text
frontend/src/
  app/layout
  app/providers
  app/store
  entities/model
  entities/section
  features/import-dxf
  features/import-json
  features/export-json
  features/selection
  features/view-modes
  widgets/viewport-3d
  widgets/project-tree
  widgets/properties-panel
  shared/math
  shared/ui
  shared/utils
```

Добавь пустые `index.ts` там, где это уместно.

## Проверки
- `cd frontend && npm run build`
- `cd frontend && npm run lint`
