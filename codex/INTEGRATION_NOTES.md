# Интеграционные заметки

## Назначение

Этот файл фиксирует, как был интегрирован расширенный frontend/UI/console блок задач после завершённой `9.2`, без потери старых невыполненных задач по real-view, results и docs.

## Итоговая схема

1. Задачи `10.1–17.2` заняты новой UI/model/console итерацией.
2. Старые невыполненные задачи `10.1–12.2` перенесены в диапазон `18.1–20.2`.
3. Основной источник правды по порядку и статусам: `codex/IMPLEMENTATION_PLAN.md`.
4. Machine-readable список задач: `codex/TASKS.json`.
5. Канонические docs по данным и импорту сведены к:
   - `codex/docs/grideng-model-v0.2.md`
   - `codex/docs/dxf-import-spec-v0.1.md`

## Перенумерация
